# teacoder.github.io
[teacoder.github.io](https://teacoder.github.io)

My small static portfolio site built with **Markdeep**.  Pages are simple HTML wrappers
containing Markdeep source; the navigation bar and footer are shared via
`navbar.html`/`footer.html` and injected at runtime by small scripts in
`/scripts`.

## Features
- Header with three sections: **Blog**, **Projects**, **About**
- Blog index reads `posts/post.json` and lists `.md.html` entries stored in
  `posts/`
